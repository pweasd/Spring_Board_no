# Spring WebProject01 - MVC Board

Spring을 통해 만든 게시판 제작과정과 기능설명 및 설정

## 0. 개발환경
해당 프로젝트에서는 오라클 가상머신에 설치된 리눅스서버와 윈도우에 설치된 Tomcat7을 연동하여 게시판을 제작한다. 
|                |                     	 |
|----------------|-------------------------------|
|Os				 |`Windows10 & Linux(Ubuntu)`    |
|DataBase        |`Mybatis & Mysql`              |
|FrameWork       |`Spring-MVC`                   |
|Tool          	 |`Eclipse EE`                   |
|WAS          	 |`Tomcat7`                   |
|Lib          	 |`Mybatis`                   |
